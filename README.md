## 📋 About

A clean, professional website template designed for a construction or contracting company. Built entirely with HTML and CSS — no JavaScript frameworks, no dependencies. This project demonstrates the ability to deliver polished, real-world client websites using only core web fundamentals.

This is the kind of site small businesses and contractors across Nigeria need — and it shows I can build and deliver it.

---

## ✨ Features

- Fully responsive — works on mobile, tablet, and desktop
- Professional layout suited for a real business client
- Clean multi-section single page design
- CSS-heavy — demonstrates strong layout and styling skills
- No JavaScript dependencies — pure HTML and CSS
- Fast loading — lightweight with no frameworks
- Deployed live on Vercel

## 🚀 Getting Started

### View it live
👉 **[construction-website-rose-delta.vercel.app](https://construction-website-rose-delta.vercel.app)**

### Run locally

```bash
# Clone the repo
git clone https://github.com/Eldren-cmd/construction-website.git

# Open in your browser
cd construction-website
open index.html
```

No build step, no installs needed — just open `index.html` directly in your browser.

## 🧹 Note on Repository Hygiene

This repo currently has `node_modules/` and `.vscode/` committed. These should not be in version control. If you are forking or cloning this project, add a `.gitignore` with:

```
node_modules/
.vscode/
```
